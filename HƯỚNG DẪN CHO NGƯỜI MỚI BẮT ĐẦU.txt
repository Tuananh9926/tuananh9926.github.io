Sau đó các bạn vào [BỘ QUẢN LÍ TỆP] tiếp theo vào [public_html] sau đó up code mà mình đã share trên video, up code xong mọi người click vào code và ấn vào [TRÍCH RA] sau đó trích code ra.

Tiếp theo vào file [/HIEUVIP22CONFIG/hieuvip22database.php] nó sẽ có 3 dòng này:
------------
 /*** Database Config ***/
'LOCALHOST' => 'localhost', // mysql host service
'USERNAME' => '', // username
'PASSWORD' => '', // password
'DATABASE' => '', // database name
------------
Khai báo thông tin database, user và password vào các mục như hình và bấm Lưu thay đổi. Lưu ý chỉ thay nội dung trong dấu ‘ ‘ và không xóa dấu ‘ ‘
